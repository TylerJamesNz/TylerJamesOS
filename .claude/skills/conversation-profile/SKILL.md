# Conversation profile — Tyler

## Tone
- Short, matter-of-fact, casual Australian
- **Capitalise the first letter of the message.** Rest casual.
- Minimal punctuation, no exclamation marks
- No preamble, no sign-offs ("cheers"/"thanks"/"Tyler") — bare
- No emoji unless explicitly requested

Note: Tyler's own chat TO Claude is often fully lowercase/typo-ish. The messages Claude drafts for Tyler to SEND are capitalised at the start. Don't mirror his typing back.

## Greetings
- "Hey team" — flatmates, group
- "Hey guys" — casual group
- "Hey [name]" — one-on-one

## Bill-split template (flatmates)

Prefer, when period is known:
`Hey team [bill type] bill is $[total] for [period] so $[per-person] each`

Fallback, when period not known (e.g. hidden in PDF):
`Hey team latest [bill type] bill is $[total] so $[per-person] each`

Never insert a placeholder like `[period]` or `for ___`. Either
include a real period or drop it and use "latest".

Examples:
- `Hey team hot water bill is $90 for May so $30 each`
- `Hey team latest electricity bill is $380.47 so $126.82 each`

Flatmates: Lucy, Matt (3-way split including Tyler)

## Don'ts
- Corporate phrasing ("I hope this finds you well", "please find attached")
- Preamble before the ask
- Sign-offs
- Emoji
- Placeholders like [X] left in the output

## When in doubt
Show Tyler the draft before sending. Ask for a sample if context is unfamiliar.
